TEAM Gummy
==============

Gummy AOSP 4.4.2

Sync and Build:
---------------

In terminal enter:

    {x}   repo init -u git://github.com/dave71670/gummy_platform_manifest.git -b kk4.4
    {x}   repo sync
    {x}   prebuilts/misc/linux-x86/ccache/ccache -M 50G
    {x}   . build/envsetup.sh && brunch "device" -j8
